# space-invaders 

Simple 2D-game written in Go using [Ebiten](https://github.com/hajimehoshi/ebiten).

## Build&Run

```bash
git clone https://github.com/dstdfx/space-invaders
cd space-invaders && go build .
./space-invaders
```
